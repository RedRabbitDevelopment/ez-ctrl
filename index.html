EZ Controller

A lightweight, simple nodejs framework.

Download .zip Download .tar.gz View on {{site.githubUrl}}GitHub

Write it once, use it everywhere.

Once you have a Controller configured, you can set it up to work over HTTP, Socket.io, or from another place within your code.

Easy client-side/server-side validation

For security reasons, you should always validate/cleanse user input on the server-side, but additional client-side checking increases speed. With EZ-Controller, your validation configuration is sent downstream, so you only have to configure the validation once, and we'll validate the input on both ends!

Automatically generate client-side/server-side libraries

With your server already aware of the routing configuration, why not generate some easy-to-use libraries to use on your client? With the gulp-ez and grunt-ez libraries, you can easily package a javascript file to deploy onto your client or use on other servers.
